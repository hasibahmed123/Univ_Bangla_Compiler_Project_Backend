পরীক্ষা 2: যদি-নাহলে (usable for > , < , ==)

ইনপুট: যদি (২ > ১) { লেখ "সঠিক"; } নাহলে { লেখ "ভুল"; }
ইনপুট: যদি (৩ == ৩) { লেখ "মিলে গেছে"; } নাহলে { লেখ "মেলেনি"; }
